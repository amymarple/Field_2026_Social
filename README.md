# Field_2026_Social
This repository is designed for preprocessing and analyzing data from Field_2026

## Folder Structure

- `/preprocessing`
  - `/file_copy_checking` - verify copied files are complete and consistent.
  - `/storage_checking` - validate storage availability and dataset integrity.
  - `/video_merge` - merge videos collected from different cameras.
  - `/animal_position_tracking` - track animal positions after merged video preparation.
- `/analyzing` - store analysis scripts and outputs derived from preprocessing results.
